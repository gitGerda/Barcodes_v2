NiceLabel SDK.NET DemoApp Project

What is it?
-----------

The NiceLabel SDK.NET DemoApp is a C# WPF project that 
demonstrates using many of the SDK features.  This project
shows one way of using the SDK with databinding and the
Model-View-ViewMdoel (MVVM) pattern.  


Key Examples
----------
The intent of this project is to demonstrate what can be done
as a complete implementation of using the SDK.  This is not intended
to be used as a learning tool.  Please see the other sample applications
that contain targeted usage examples.


More Information
----------------

For complete instructions on using the NiceLabel SDK.NET, please see 
NiceLabel SDK.NET Installation and Deployment Guide.
